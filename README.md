# 🎨 Collabboard

A real-time collaborative whiteboard inspired by Excalidraw that enables teams, students, and creators to brainstorm, sketch, and collaborate together on a shared canvas. Built with modern web technologies and designed for seamless multiplayer interaction.

🔗 **GitHub Repository:** https://github.com/GitankRana/Collabboard

---

## 🛠️ Tech Stack

| Layer            | Technology                                  |
| ---------------- | ------------------------------------------- |
| Frontend         | React, TypeScript, Vite                     |
| State Management | Zustand                                     |
| Backend          | Node.js, Express                            |
| Realtime         | WebSockets                                  |
| Styling          | CSS / Tailwind CSS                          |
| Deployment       | Vercel (Frontend), Railway/Render (Backend) |

---

## ✨ Features

* 🎨 Infinite collaborative whiteboard
* 👥 Real-time multiplayer synchronization
* 🔗 Room-based collaboration via shareable links
* 🖱️ Live cursor presence tracking
* ⚡ Low-latency WebSocket communication
* 📱 Responsive design for desktop and mobile
* 🔄 Real-time canvas updates across connected users
* 🚪 Create and join collaboration rooms instantly
* 💾 Future support for persistent boards and history
* 🌙 Clean modern UI inspired by Excalidraw
